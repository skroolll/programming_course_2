## Как студенту работать

1. Сделать **fork** этого репозитория к себе на GitHub.
2. Склонировать свой fork в Visual Studio: *Clone a repository*.
3. Открыть `cpp-course.sln`.
4. Выбрать нужный проект `LabXX` → *Set as Startup Project*.
5. Запуск: **Ctrl+F5** (без отладки).

## Общий стандарт вывода
- выводим **без лишних пояснений**;
- числа — через пробел;
- если нужно несколько строк — печатаем в **фиксированном порядке**.

## Сдача
- На защите студент объясняет код и отвечает на вопросы.
- Дедлайн контролируется по времени **последнего commit/push** в fork.
